# Visual System

No formal asset kit yet. These guidelines define the visual direction.

## Visual Traits
- Calm, monochrome-leaning palette
- Minimal, editorial layout
- Clear hierarchy with ample spacing

## Color Direction
- Base: charcoal, slate, and off-white
- Accent: muted amber or desaturated blue
- Avoid neon or saturated gradients

## Typography Direction
- Headings: a modern serif for authority
- Body: a clean sans-serif for clarity
- Avoid playful display fonts

## Layout Guidance
- Content-first, minimal ornamentation
- Use rules and dividers to separate sections
- Prefer long-form readability

## Iconography
- Simple line icons only
- Avoid emoji in formal brand materials
